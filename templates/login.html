Cómo hacer login
  1. Abre una terminal en el directorio del proyecto.
  2. Ejecuta el script de login:
    python login_instagram.py
  3. Se abrirá una ventana de Chrome. Inicia sesión en Instagram normalmente (usuario y contraseña, 2FA si aplica).
  4. Cuando estés dentro del feed, vuelve a la terminal y presiona Enter.
  5. El estado se guarda en ig_state.json y el scraper lo usará automáticamente.
Consejo: Si el scraper empieza a fallar o devuelve 0 posts, repite el proceso de login — las sesiones de Instagram expiran.
